# CUSTOMER-SERVICE-REQUEST-ANALYSIS-PowerBi
Customer Service Request Analysis Dashboard (Power BI) This project presents a comprehensive analysis of customer service requests using Power BI. The dashboard is designed to explore complaint patterns, identify major complaint types, and evaluate service efficiency across different cities.
📊 Project Overview
This project focuses on analyzing customer service requests using Power BI to identify patterns, major complaint types, and response efficiency across different cities.
🎯 Objectives
Analyze complaint frequency across cities
Identify major complaint types
Evaluate response time performance
Visualize complaint distribution geographically
Perform statistical analysis for insights
📈 Key Features
KPI Cards (Total Complaints, Closure Rate, etc.)
Complaints by City (Bar Chart)
Top Complaint Types
Geographic Map Visualization
Response Time Analysis
Interactive Filters (City, Complaint Type, Date)
📊 Tools Used
Power BI
DAX (Data Analysis Expressions)
Data Cleaning & Transformation
🔍 Key Insights
Most complaints are concentrated in specific cities like Brooklyn
Certain complaint types (e.g., Blocked Driveway) occur more frequently
Response time varies depending on complaint type
High closure rate indicates efficient service handling.
